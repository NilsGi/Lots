Tack för besöket på SecTech 2025!

Vi på LOTS Security vill rikta ett stort tack till alla som besökte vår monter på SecTech-mässan.

Det var givande att möta så många engagerade besökare med fokus på säkerhet – både befintliga kunder och nya kontakter i branschen.

Vi uppskattar det stora intresset för våra säkerhetslösningar och larmsystem, och ser fram emot att fortsätta dialogen och utforska framtida samarbeten.

Missade du oss på plats?
Du är alltid välkommen att kontakta oss för att få veta mer om hur våra system kan bidra till ökad trygghet i din eller er kunds verksamhet.

👉 Hör av dig till oss på LOTS Security – vi hjälper dig gärna!

Våra säkerhetslösningar:

  • Personlarmsystem med positionering inomhus och utomhus
  • Enklare bråk-/personlarm med trådlösa knappar
  • Ensamarbetarlarm med tvåvägskommunikation
  • Trådbundna och trådlösa kallelsesignalsystem
  • Nödlarmsystem för simhallar
  • Hjälpknappar för butiker
  • Cronapress – larmlist för högriskområden
