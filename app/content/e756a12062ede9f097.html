NurseCare – Server

NurseCare server

Har förinstallerat grundprogramvara, server med fläktlös chassi (tyst)
Duplicerat dataminne. Inkluderar operativsystem och SQL-databasprogramvaran.
Inkluderar automatiskt dataskydd.

Servern finns i tre olika storlekar

Small = 25 rum

Medium= 70 rum

Large= 300 rum

Innehållet i servrarna styrs via olika licenser (engångskostnad) och styrs av vilka behov av funktioner som önskas.

Licenser som behövs för olika funktioner

NurseLog Basic

Operatör och administrationsprogram NurseLog Basic för upp till 2 avdelningar och max
100 användare/rum, inklusive:
■ Automatisk säkerhetskopiering av data
■ SQL-databas
■ Möjlighet för fjärruppkoppling via IP

NurseLog Professional

Operatör och administrationsprogram NurseLog Professional (virtuell server) för ett obegränsat antal
avdelningar och brukare/rum inkl
■ Automatisk säkerhetskopiering av data
■ SQL-databas
■ Möjlighet för fjärruppkoppling via IP

NurseLog Always

Ett duplicerat NurseLog-cystem för nästan 100 % hög tillgänglighet.
Inkluderar mjukvara och hårdvara.
NurseLog Wireless input protokoll

Serverlicens för användning av NurseLog hårdvara “NurseCare Wireless Receiver (IPWR)” (artikel ERTH0035).
En licens för ett obegränsat antal stationära och mobila trådlösa enheter.

NurseLog-Maps

Programvara för att visa oriteringsritning för varje område eller våning som visar aktiva händelser i varje sektion som visas. Till exempel blinkar och
ljudaviseringar om boendes anrop, assistans och akut anrop,

InfoTab information

Grundläggande licens för att visa information på vägg & tak displayer.
Den kan visa till exempel information om väder, aktiviteter i anläggningen, nyheter, lunchmenyer eller skräddarsydda
webbplatser. Den integrerade mjukvaran gör det möjligt för användarna att presentera önskad information utan behov
av HTML-kunskaper.
