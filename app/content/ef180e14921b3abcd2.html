GateWay för koppling av data till Intercall Cloud
Lagra data och informerar oc fakta. Intercall Cloud -tjänsten används för säker fjärråtkomst till Ert Intercall -system. Lämplig för alla vårdgivare som behöver har kontroll på alla vård och omsorgshändelser över tid. Intercall Cloud samlar och sammanställer dina data till en enda åtkomstpunkt där händelser kan analyseras och skapa rapporter för redovisning.
