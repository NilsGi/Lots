Rumsapparat Basic

Touch Rumsapparat Basic vår enklaste anropsapparat, idealisk för användning i de flesta vårdmiljöer. Placeras bredvid sängen eller på allmänna platser, såsom väntrum och korridorer m.m. Ett flertal programmerbara larmnivåer kan användas för att indikera vilken typ av larm och vård som krävs. Rumsapparat Basic har utöver den innovativa dubbeltrycks-funktionen vid akutläge även ett Kod Blå-alternativ.

Fördelar med Rumsapparat Basic


Flera larmnivåer

Rumsapparaten Basic har flera programmerbara larmnivåer, inklusive anrop, larmnärvaro, prioritet, assistans och kod blå. Dessa kan användas för att identifiera olika typer av larm  exempelvis om den boende/patienten har fallit ur sängen eller helt enkelt bara vill ställa en fråga till vårdpersonalen.


Akutläge

Rumsapparaten Basic har utöver den innovativa dubbeltrycks-akutfunktionen även ett Kod Blå-alternativ som kan vara avgörande på hjärt- och intensivvårdsavdelningar.


Flexibel uppkoppling

Rumsapparaten Basic har installerats på det enkla Intercall Bus nätverket och kan fungera som det centrala nav som gör det möjligt för uppkoppling mot avlägsen apparatur såsom dragsnören, tryckmattor och päronkontakter.


Trygghet

Rumsapparat Basic har en inbyggd LED-indikator, som ger patienten eller den boende en omedelbar visuell indikation på att deras anrop har skickats. En ljudsignal informerar sjuksköterskan om att det finns ett aktivt larm i systemet.

Flertal larmnivåer

Rumsapparaten Basic har flera programmerbara larmnivåer, inklusive anrop, mat, larmnärvaro, prioritet, assistans och kod blå. Dessa kan användas för att identifiera om patienten eller den boende behöver hjälp vid toalettbesök, är hungrig eller törstig, har fallit ur sängen eller helt enkelt kräver uppmärksamhet.
