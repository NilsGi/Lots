# LOTS Security – fristående webbplats

Den här leveransen är helt statisk och använder inte WordPress, PHP, databas eller något JavaScript-ramverk.

## Publicera

1. Ta en fullständig backup av nuvarande webbrot.
2. Behåll katalogen `/wp-content/uploads/` eftersom den innehåller LOTS befintliga bilder och dokument. Katalogen används enbart som statiskt filarkiv; WordPress behöver inte vara installerat eller aktivt.
3. Ladda upp allt innehåll i denna leverans till domänens webbrot.
4. Kontrollera startsida, produktsökning, nyhetsarkiv och ett urval undersidor.
5. Koppla formulär, nyhetsbrevsregistrering, kundinloggning och eventuell varukorg till de nya fristående tjänster som LOTS väljer innan skarp lansering.
